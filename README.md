# 🧘 Zen Timer Pro

A minimal, high-performance web-based productivity timer designed for focus. Zen Timer Pro combines a sleek aesthetic with powerful features like haptic feedback, auto-dimming, and session tracking.



## ✨ Features

* **Dual Modes:** Seamlessly toggle between a classic **Stopwatch** and a customizable **Pomodoro** timer.
* **Immersive Design:** Supports custom background colors, font styles, and **Wallpaper Uploads** with automatic text-shadow adjustment for readability.
* **Smart Dimming:** Optional "Auto-Dim" feature kicks in after 2 minutes of activity to reduce screen glare and save battery.
* **Haptic & Audio:** Integrated "Wind Chime" sound for session completion and adjustable haptic feedback for touch interactions.
* **History & Analytics:** Tracks your sessions locally and provides a 7-day daily activity breakdown.
* **Privacy First:** All data is stored locally in your browser's `localStorage`. No tracking, no cookies, no backend.

## ⌨️ Keyboard Shortcuts

Master your workflow with these quick keys:

| Key | Action |
| :--- | :--- |
| `Space` | Start / Stop Timer |
| `R` | Reset Timer / Log Session |
| `M` | Toggle Mode (Stopwatch/Pomo) |
| `S` | Open Settings |
| `H` | Open History & Stats |
| `F` | Toggle Fullscreen |

## 📱 Mobile Use

Zen Timer Pro is fully responsive and designed to feel like a native app.
1. Open the live link in your mobile browser.
2. Select **"Add to Home Screen"** from your browser menu.
3. Launch from your home screen for a distraction-free, full-screen experience.

## 🛠️ Customization

You can personalize the following via the Settings panel:
- **Visuals:** Background color, Timer color, and 4 distinct typography styles.
- **Feedback:** Toggle Wind Chime sounds and adjust Haptic intensity.
- **Scale:** Adjust the timer size to fit any display, from mobile phones to ultra-wide monitors.
